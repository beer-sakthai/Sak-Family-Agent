# lego-brick

Lego-brick style with interlocking studs, blocky layers, and bright primary colors. Great for kids' content, building blocks, and playful branding.

## Mood Board

- Primary: #FEFEEF <off-white> - background
- Bricks: #E32F2F <red>, #006DA7 <blue>, #FFD500 <yellow>, #2AD138 <green>, #FFFFFF <white>
- Text: #222222 (dark grey) - primary content
- Medium: #DCDCDC (light grey) - support

## Fonts

- Headings: Bold, sans-serif (e.g., Fairplay Bold, Lego Sans)
- Body: Clean sans-serif, rounded

## Style Traits

- Texture: Flat, blocky, with brick nochs on top
- Illustrations: Lego studs, brick shapes, building blocks
- Layout: Subtle grid, sectioned, building-up style
