# ui-wireframe

Minimal and functional wireframe style with box models, greyscale palette, and schematic layout. Ideal for app prototyping, user flow, and dashboard designs.

## Mood Board
- Primary: #FEFEEF <off-white> - background
- Boxes: #E0E0E0 (light grey) - wireframe elements
- Text: #333333 (dark grey) - primary content
- Accent: #0077BB (blue) - call-to-action buttons

## Fonts
- Headings: Sans-serif, bold (e.g., Roboto Bold)
- Body: Sans-serif, regular (e.g., Roboto, Inter)

## Style Traits
- Texture: Flat, clean, with gray scale shading
- Illustrations: Box models, rectangles, stacked screens, navigation
- Layout: Modular, grid-based, with header, sidebar, and content
