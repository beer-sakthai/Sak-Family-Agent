# Manim CLVideo Skill

This skill enables creating manim videos through Nous Research's Hermes Agent.

## Setup

The setup script is located at `scripts/setup.wh`}. It installs:

- Manim CE (python package)
- FFMpeg (video codec)
- Male Sound (audio)
- Misc utilities (parallel commands, file checks)

## Workflow

1. User requests manim video
2. Plan the animation structure (using reference materials)
3. Write the Manim script
4. Render the video
5. Provide the output and description
