# winding-roadmap

A winding path narrating a process, experience, or journey with twists and turns.

## Structure

- Starting point
- Winding road or path
- Stops or way points
- Destination or goal
- Optional detours
- Background elements

## Best For
- User journeys/experiences
- Narrative processes
+ Learning paths
-                      travel routes
-                      program pathmays

## Visual Elements
- Curvilinear path
- Stop markers along the way
- Arrows indicating direction
- Landmarks or landscape elements
- Start/end icons

## Text Placement
- Title at top
- Labeled way points
- Descriptions at each stop
- Optional alternative path labels

## Recommended Pairings
- `cartoon-hand-drawn`: Engaging journeys
- `storybook-watercolor`: Dreamy experience maps
