# venn-diagram

Overlapping circles showing shared and unique aspects.

## Structure

- 2 + circles or ellipses
- Overlap zone for shared qualities
- Unique sections for differences
- Sizing can represent weight/importance

## Best For
- Comparative analysis
- Tasks/skills overlap
- Set difference
- Intersection analysis

## Visual Elements

- Two or more circles
-Overlapping region
- Color fill for each set
- Overlap color blend
- Optional dotted borders for subsets

## Text Placement
- Title at top
- Set names or inside areas
- Overlap description in the middle
- Optional characteristics at the bottom

## Recommended Pairings
- `palette-chikashi`: Insightful comparisons
- `corporate-memphis`: Business comparisons with flavor
