# periodic-table

Grid of elements governed by categories, dimensions, or taxonomies.

## Structure

- Table-like grid
[§ Rows and columns
- Category labels for each dimension
- Entries at intersection points
- Application-to-list or display

## Variants

| Variant | Focus | Visual Emphasis |
|---------|--------|-----------------|
| **Classic** | Items in categories | Rows & columns |
| **Six Cells** | Complex relationships | Grid with big intersection points |
| **Comparison** | Analysis of items | Rows/columns with values |

## Text Placement
- Title at top
- Column names atop each column
- Row names at left of each row
- Cell content (short, dense)
- Legend if needed

- `cartoon-hand-drawn`: Family-friendly categorization
