# structural-breakdown

Displaying the hierarchical structure of an organization, system, or concept.

## Structure

- Top-level container
- Main categories below
- Sub-categories under each
- Connectors between sections
- Optional depth linit

## Best For
- Org charts/agency structures
- System modules/components
- Concept hierarchies
- Information architecture

## Visual Elements

- Nested rectangles or boxes
- Level indentation
- Connecting lines
- Title bars per section
- Expand/collapse indicators

## Text Placement
- Title at top
- Section headers
- Sub-section labels
- Short descriptions
- Optional detail on demand

## Recommended Pairings
- `technical-schematic`: Precise system architecture
- `hand-drawn-edu`: Educational concept breakdowns
