# Animation Design Thinking - Reference

## How to Approach Animation Design

This guide outlines the design thinking process for creating manim videos in the style of 3Blue1Brown.

1.  _Start with a script_: Write a full script or outline before animating. This is the most important step.
2.  **Identify key concepts**: What are the main mental models or ideas that need visual representation?
3.  **Plan the scenes** Break down the explanation into segments:
    - Problem Setup / Introduction
    - Build up / Explanation
    - Examples / Visualizations
    - Summary / Conclusion
4.  **Design animation transitions**: How do we move between concepts?
    - Transformation (transform): Move from one state to another
    - Episodic (write): Reveal information or hold attention
    - Build (create): Show the construction of complex ideas
5.  **Plan for visual variation**: Don't just show text. Use graphs, diagrams, and analogies to make the explanation more intuitive.
