# story-mountain

Climactic structure building toward a climax or outcome.

## Structure

- Base/starting point
- Rising action/conflict
- Climax turning point
- Falling action/resolution
- Endpoint/resolution

## Best For

- Story/narrative flows
- Issue/solution frameworks
- Project paths
- User Journeys
- Case studies

## Visual Elements

- Mountain shape outline
- Labeled points on the curve
- Arrows along the path
- Highlights at peak
- Optional images at steps

## Text Placement
- Title above our at top
- Each point labeled
- Brief explanation below or beside
- Goal/result at the end

## Recommended Pairings
- `cartoon-hand-drawn`: Engaging user journeys
- `bold-graphic`: Outcome-focused progress maps
