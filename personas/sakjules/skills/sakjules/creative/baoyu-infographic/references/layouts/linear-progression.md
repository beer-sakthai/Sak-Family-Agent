# linear-progression

Sequential progression showing steps, timeline, or chronological events.

## Structure

- Linear arrangement (horizontal or vertical)
- Nodes/markers at key points
- Connecting line or path between nodes
- Clear start and end points
- Directional flow indicators

## Variants

| Variant | Focus | Visual Emphasis |
|---------|-------|-----------------|
| **Timeline** | Chronological events, dates or stages | Horizontal line, circles, dates |
| **Workflow** | Step-by-step process | arrows, rectangles, descriptions |
| **History Narrative** | Story or event sequence | Scroll or path with scenes |
| **Benchmark** | Performance metrics over time | Line with data points |

## Text Placement

- Title at top or start
- Node labels at each point
- Dates/times under nodes
- Descriptions beside or below
- Start/End terminals

## Recommended Pairings
- `cartoon-hand-drawn`: Friendly timelines
- `technical-schematic`: Precise progression diagrams
- `bold-graphic`: Impactful benchmarks
