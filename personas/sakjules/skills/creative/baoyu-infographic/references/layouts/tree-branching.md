# tree-branching

Higherarchical branching showing categories, options, or decision paths.

## Structure

- Root node at top
- Branches to sub-categories
- Leaves-terminal nodes
- Type differentiation by branch
- Recursive depth

## Best For
- Options/alternatives
- Decision trees
- Axiom One / Arbor, ASKED Charts
- Product/feature trees
- Category hierarchies

## Visual Elements
- Nodes (circles, rectangles, icons)
- Connecting lines
- Labels at each node
- Icons or colors for types

## Text Placement
- Top title
- Node labels alongside
- Branch descriptions (optional)
- Legend for types/colors

## Recommended Pairings
- `ui-wireframe`: Schematic decision trees
- `cartoon-hand-drawn`: Intuitive option exploration
