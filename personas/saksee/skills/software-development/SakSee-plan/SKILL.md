# Plan Mode

Write a detailed execution plan before building.

## Format

Output to `.hermes/plans/` as markdown:

```markdown
# Plan: [Title]

## Goals
- ...

## Steps
1. ...
2. ...

## Files to Change
- path/to/file: what to do

## Risks
- ...
```

## Rules
- Bite-sized tasks
- Exact file paths
- Complete code, not pseudo-code
