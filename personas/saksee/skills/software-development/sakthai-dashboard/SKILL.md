# SakThai Dashboard

Set up and run the SakThai Agent dashboard.

## Location

- Source: `beer-sakthai/Sak-Family-Agent/dashboard/`
- Deployed on GitHub

## Setup

```bash
cd dashboard
pip install -r requirements.txt
python app.py
```

## Features

- Agent status monitoring
- Memory browser
- Conversation history
- Skill management
- Configuration editor
