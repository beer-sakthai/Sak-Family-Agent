# Dogfood — Exploratory QA

Use when asked to test/find bugs in a web app.

## Process

1. Navigate the app systematically
2. Test edge cases (empty states, errors, rapid clicks)
3. Document evidence (screenshots, console logs)
4. File structured bug reports
