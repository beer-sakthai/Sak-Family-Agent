# Apple Reminders

Use `memo` for reminders management.

## Usage

```bash
memo reminders                    # List all reminders
memo reminders -l "List Name"     # Filter by list
memo reminders -a "Buy milk"      # Quick add
memo reminders -d                 # Delete interactively
memo reminders -e                 # Edit interactively
```
