# SakThai 6-Cycle SFT Dataset

A small, synthetic chat-format dataset for supervised fine-tuning of the `Nanthasit/sakthai-coder-3b` coding assistant.

## Purpose

Teach the model to respond through the SakThai six-stage cycle:

1. **Dream** — understand intent and success criteria
2. **Hope** — propose a practical, positive plan
3. **Care** — consider edge cases, safety, and context
4. **Joy** — implement cleanly and explain clearly
5. **Trust** — verify, test, and state assumptions
6. **Growth** — reflect and suggest a next step

## Format

Each example is a JSON object with a `messages` array (OpenAI-style chat) and `metadata`:

```json
{
  "messages": [
    {"role": "system", "content": "You are SakThai..."},
    {"role": "user", "content": "Write a Python function..."},
    {"role": "assistant", "content": "**Dream**: ... **Growth**: ..."}
  ],
  "metadata": {
    "task_type": "code_generation",
    "cycle": ["Dream", "Hope", "Care", "Joy", "Trust", "Growth"],
    "source": "synthetic",
    "example_id": 0
  }
}
```

## Task types

- `code_generation`
- `bug_fixing`
- `code_review`
- `architecture`
- `refactoring`
- `explanation`
- `cycle_practice`

## Usage

```python
from datasets import load_dataset

ds = load_dataset("Nanthasit/sakthai-cycle-6-sft", split="train")
```

## Intended fine-tune target

- Base model: [`Nanthasit/sakthai-coder-3b`](https://huggingface.co/Nanthasit/sakthai-coder-3b)
- Training framework: TRL `SFTTrainer`
