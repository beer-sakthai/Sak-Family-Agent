# Jules Ecosystem Journals

Welcome to the `.Jules` directory. These files are specialized journals kept by the sub-roles of the **SakJules** persona (the automation and CI/CD master).

## Sub-Roles & Journals

- **Bolt** (`bolt.md`): Focuses on SQL optimization, database performance, and backend efficiency.
- **Palette** (`palette.md`): Focuses on UI/UX, dashboard accessibility, and design consistency.
- **Sentinel** (`sentinel.md`): Focuses on security, vulnerability prevention, and defense-in-depth patterns.

## Format Convention

All journals follow a consistent format for entries:

```markdown
## YYYY-MM-DD - [Topic]
**[Context/Vulnerability]:** Brief description of the background or issue.
**Learning:** What was learned from the experience.
**[Action/Prevention]:** Actionable steps to prevent or apply the learning.
```
