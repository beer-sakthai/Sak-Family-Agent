## Option A — git subtree pull (recommended)

From inside the `sakthai-agent-v2` monorepo root:

```bash
# Pull all new commits from the standalone repo into infra/hermes-agents/
git subtree pull \
  --prefix=infra/hermes-agents \
  git@github.com:beer-sakthai/sakthai-hermes-agents.git \
  main \
  --squash \
  -m "sync: hermes-agents 6 commits from 2026-06-29 session (voices, deploy, skills)"
```

> Note: First push the standalone repo (`cd ~/sakthai-hermes-agents && git push`)
> so the remote is up to date before running subtree pull.

## Option B — manual patch apply

From the monorepo root:

```bash
# Apply the 6 patches (generated by git format-patch) with prefix rewrite
for patch in /path/to/patches/000{1..6}-*.patch; do
  git am --directory=infra/hermes-agents "$patch"
done
```
